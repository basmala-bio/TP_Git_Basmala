Rapport du TP Git & GitHub
Boubakker Basmala
Spécialité : Toxicologie fondamentale appliquée
Résumer:
Git est un outil très utile car il permet de suivre l’évolution d’un projet et de conserver toutes les versions de manière organisée.
Dans ce TP, j’ai appris à créer un dépôt Git, faire des commits, travailler avec Google Colab et envoyer les fichiers vers GitHub.
Mon idée de projet personnel est de créer un petit système pour enregistrer et suivre les données d’expériences de laboratoire, en utilisant Git pour garder l’historique des modifications.
