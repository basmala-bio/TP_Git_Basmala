Rapport du TP Git & GitHub
Boubakker Basmala
Spécialité : Toxicologie fondamentale appliquée
Résumé :
Git est un outil important qui permet de suivre l’évolution d’un projet, enregistrer les différentes versions et travailler efficacement en groupe. Dans ce TP, j’ai appris à créer un dépôt Git, faire des commits, configurer Git dans Google Colab et envoyer le travail vers GitHub. J’ai aussi appris à créer une branche de développement (develop) pour travailler séparément de la branche principale.
Projet biotechnologique :
Je voudrais développer un projet biotechnologique qui consiste à créer une petite base de données pour suivre les résultats d’expériences de laboratoire (croissance de bactéries, analyses de protéines). Git me permettrait de sauvegarder l’historique des modifications et de travailler proprement sur différentes versions.
